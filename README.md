# SkillCraft Technology – Task 4 (To-Do List)
This repository contains the submission for Task 4 of the SkillCraft Technology internship.
## Project: To-Do List Web Application
A simple and interactive to-do list application that helps users manage daily tasks efficiently.  
Users can add, mark, and remove tasks in an easy-to-use interface.
## Features
- Add new tasks
- Mark tasks as completed
- Delete tasks
- Clear and responsive UI
- Real-time task updates
## Technologies Used
- HTML
- CSS
- JavaScript
## Description
This project demonstrates the use of JavaScript for handling user input, DOM manipulation, and basic application logic, along with HTML for structure and CSS for styling.
